IoT Project

Smart Float

It is an smart float, which through WiFi connection provides data from its sensors to a database Crate, which manages alerts and historical values. In addition, by obtaining the local time using a NTP server performs daily the addition of the chlorine measurement corresponding to the volume of the pool.
As an additional item to highlight, in the realization of the microcontroller code to perform all these functions, it was aided by the recently released “GitHub Copilot” tool, which saved a lot of time in the development of the project.

Link to the repo!

Tecnologies used include:

  • ESP microcontroller
  • Graphana Database (SQL)
  • C programming language
